# Characters

Drop character images in **this folder** and they automatically become
selectable fighters in Roll the Brawl (and future games).

- Supported: `.png`, `.jpg`, `.jpeg`, `.webp`, `.gif`, `.svg`
- **Transparent PNGs look best** (no background box around your fighter).
- The name shown in the game comes from the filename:
  `ninja-ken.png` → **Ninja Ken**, `lava_dragon.png` → **Lava Dragon**.
- Square-ish images fit best. Make them **face to the right** ➡️ — the game
  mirrors the right-hand fighter so the two always face each other.

No restart needed — added files appear next time a game starts.
